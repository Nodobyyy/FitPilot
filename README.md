# FitPilot - 智能健身导航助手 🌿

FitPilot 是一款纯前端实现的轻量级智能健身与健康管理 Web 应用。您的所有数据均通过浏览器原生的 Web Crypto API 进行了高强度 AES 加密，并安全地保存在本地磁盘文件 (`fitpilot_data.enc`) 中。通过接入最新的视觉多模态大语言模型（如 Qwen-VL），FitPilot 将为您提供前所未有的全自动化、智能化的健康追踪体验。

## ✨ 现有核心功能

### 1. 📊 总览与深度 AI 分析
- **目标追踪系统**：半环形仪表盘直观展示距离目标体重的进度。
- **智能热量对比**：基于每日摄入、运动消耗以及**基础代谢（BMR）**，生成堆叠对比柱状图与目标红线，盈余还是缺口一目了然。
- **深度 AI 报告**：只需一键，AI 将根据您的目标、历史体征数据、饮食结构和运动习惯，为您生成四维深度报告（每日健康汇总、营养建议、运动策略、目标预估），配备绝美渐变主题卡片UI。

### 2. 🍽️ 饮食管理与 AI 拍照识图
- **拍照秒记**：厌烦了手动查热量？上传一张食物照片，AI 多模态大模型会自动识别食物名称并估算分量、热量（卡路里）、蛋白质、碳水和脂肪数据。
- **营养追踪图表**：直观展示今日三大营养素摄入比例与总览。

### 3. 💪 运动管理与 AI 截图识别
- **运动截图一键导入**：跑完步或健完身？只需上传 Apple Watch、Keep 或咕咚的运动总结截图，AI 会为您自动提取运动类型、时长、消耗热量和强度。
- **全景数据统计**：采用精致的毛玻璃与独立渐变色彩卡片，自动统计“本周运动总次数”、“总时长”及“总消耗”，搭配每周柱状趋势图清晰可视。

### 4. 📉 体征数据追踪
- **全方位记录**：支持记录体重、体脂率、**肌肉量、骨骼肌**、以及基础代谢（BMR）。
- **精细化趋势图**：针对体重、体脂、肌肉等提供多维度的折线图分析（其中肌肉与骨骼肌的微小变化也能通过 0.5kg 的精细刻度精准展现）。

### 5. 🎨 极致的 UI 美学体验
- **清新极简设计**：采用现代化的毛玻璃（Glassmorphism）与高透设计语言，融入丝滑的过渡动画与精美渐变色系，告别死板沉闷的传统数据图表。

## 🚀 快速启动

**Windows 用户：**
直接双击项目根目录下的 `双击运行.bat`，即可一键启动带加密存储功能的本地 Python 服务，并在浏览器中自动打开应用。

**Mac/Linux 用户：**
在终端中执行以下命令：
```bash
cd web
python server.py
```
然后在浏览器中访问：`http://localhost:8000`

## 🤖 AI 配置指南

要想体验核心的“拍照识图”与“深度健康分析”功能，只需简单配置您的 API Key：
1. 点击页面右上角的 **⚙️ AI 配置** 按钮。
2. 填入 API 接口地址与您的密钥（支持兼容 OpenAI 接口的各类大模型，例如阿里云通义千问）：
   - **Base URL**: `https://dashscope.aliyuncs.com/compatible-mode/v1`
   - **模型名称**: 推荐填写支持视觉能力的多模态模型（如 `qwen-vl-max` 或 `qwen-vl-plus`）。
3. 保存后，密钥将加密存储在您的本地浏览器中。

---

## ⚠️ 隐私与数据安全

- **本地加密存储**：您的**所有健康数据在使用 AES-GCM 高强度加密后，保存在根目录的 `fitpilot_data.enc` 文件中**，脱离浏览器缓存，即便清理浏览器也不会丢失数据。
- **防止数据泄露**：加密过程完全在本地浏览器中完成，就算别人拿到 `.enc` 文件，没有内置密钥也是一堆乱码。
- **无中间商**：API 请求直接由您的浏览器发往您配置的大模型厂商，无任何中间服务器截留您的私密数据。
